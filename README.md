# [MediaFire URL Parser](https://mediafire-url-parser.onrender.com)

## Usage
1. Copy the whole file url (https://wwww.mediafire.com/...)
2. Open your browser and go to the [app page](https://mediafire-url-parser.onrender.com)
3. Paste the WHOLE URL into the input
4. Click on the button and click on the text displayed ("SEU LINK")\
